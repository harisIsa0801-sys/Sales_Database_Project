# Sales Database Management System

This project focuses on the design and implementation of a relational database for sales data tracking. The primary objective is to demonstrate structured data management and the application of SQL logic for business data analysis.

### **My Mission**
1. **To Develop a structured schema** consisting of 6 key attributes to maintain high data integrity and organization.
2. **Implement SQL logic** to automate the calculation of total revenue by deriving values from price and quantity attributes.
3. **Optimize the database** to identify sales patterns, category performance, and revenue trends over specific time periods.

This repository serves as a practical demonstration of my SQL capabilities and database management skills in preparation for my professional internship.

---

### **How to Use This Project**
To view and run this database locally on your machine:

1. **Prerequisites:** Ensure you have **XAMPP** installed.
2. **Setup:**
   - Open XAMPP Control Panel and start **Apache** and **MySQL**.
   - Navigate to `http://localhost/phpmyadmin` in your browser.
3. **Database Creation:**
   - Create a new database named `sales_analysis`.
4. **Importing Data:**
   - Open the `sales_database.sql` file from this repository.
   - Copy the SQL code, go to the **SQL tab** in phpMyAdmin, and paste it.
   - Click **Go** to execute.
5. **Verification:** The `sales_database` table will be populated with sample data for your review.
