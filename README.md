# Sales_Database_Management_System
This project focuses on the design and implementation of a relational database for sales data tracking. The primary objective is to demonstrate structured data management and the application of SQL logic for business data analysis.

## My mission:
1)To Developed a structured schema consisting of 6 key attributes to maintain high data integrity and organization.

2)I Implemented SQL logic to automate the calculation of total revenue by deriving values from price and quantity attributes.

3)The database is optimized to identify sales patterns, category performance and revenue trends over specific time periods.

This repository serves as a practical demonstration of my SQL capabilities and database management skills in preparation for my professional internship.
